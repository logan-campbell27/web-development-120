CST-120 Web Development Syllabus

Course Description

This course introduces the fundamental concepts and syntax of the web development languages including HTML, CSS, and JavaScript. The course focuses on foundation required to build complex dynamic web applications.

Weeks 1-2

Topic 1 HTML

In this topic students are introduced to the HTML markup language where they learn basic structure, basic elements of an HTML document, and how to work with text and basic controls. The students finish this topic by learning the developer tools available to them as well as how web servers work to help support development of HTML based applications.

  1. Develop web pages with basic Elements and Structure that includes but is not limited to
    Document, Head, and Body tags.
  2. Develop web pages with Text that includes but is not limited to Headings, Paragraphs,
    Formatting, Fonts, and Color tags.
  3. Develop web pages with Components but is not limited to Links, Images, and Lists tags.
  4. Investigate how a web server functions. Identify differences between localhost, file URL and
    domain names in the URL./li>
  5. Investigate the Developers Tools available as a web page developer.

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn HTML menu under the
    HTML and CSS section. Complete all reading and tutorials under the HTML Home Tutorial
    section.

Weeks 3-4

Topic 2 Advanced HTML

In this topic students continue to expand their knowledge of the HTML markup language learning some of the more advanced features of the HTML markup language where they learn how to work with layouts, graphics, and media content. The students finish this topic by learning how to make HTML pages ADA compliant.

  1. Develop web pages with Layouts that includes but is not limited to Blocks/Div/Span, Tables, Layouts tags.
  2. Develop web pages with Graphics that includes but is not limited to Image, Canvas, and SVG tags.
  3. Develop web pages with Media that includes but is not limited to Video and Audio tags.
  4. Explain how and why ADA compliance is achieved with ADA guidelines.

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn HTML menu under the HTML and CSS section. Complete all reading and tutorials under the HTML Forms, HTML Graphics, HTML Media, and HTML API's Tutorial sections.

Weeks 5-6

Topic 3 CSS

In this topic students learn Cascading Style Sheets learn to apply styling to HTML pages using fonts, colors, and formatting. The students finish the topic by learning how to apply backgrounds, transitions, and animations to their HTML pages.

  1. Develop web pages with CSS Basics Styles that includes but is not limited to Inline, Internal, External, and Selectors styles.
  2. Develop web pages with Text that includes but is not limited to built in Fonts, custom Fonts from external sources, Color, and Alignment styles.
  3. Develop web pages with Formatting and Layouts that includes but is not limited to Backgrounds, Borders, Margins, Padding, Width/Height, and the Box Model styles.
  4. Develop web pages with that includes but is not limited to Transitions and Animations styles.

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn CSS menu under the HTML and CSS section. Complete all reading and tutorials under the CSS Tutorial and CSS Advanced Tutorial sections.

Weeks 7-8

Topic 4 JavaScript

In this topic students are introduced to the JavaScript programming language. This programming language is use as the foundation for client-side programming on web pages. Students learn the basic language syntax, how to work with variables, make decisions, perform repetitive operations, and create reusable functions.

  1. Develop code that integrates basic JavaScript (Inline and External files) in a web page.
  2. Develop code that leverages Variables and Data Types in a web pages.
  3. Develop code that leverages Operators (Math, Assignment, Comparison, Logical, Boolean, and String) in a web page.
  4. Develop code that uses Controls (If, For, Loops) in a web page.
  5. Develop code that uses reusable Functions in a web page.

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Tutorial Tutorial section.

Weeks 9-10

Topic 5 Advanced JavaScript

In this topic students continue to expand their knowledge of the JavaScript programming language learning how to create objects, work with data as arrays, handle events from DOM objects from their HTML pages. The students finish the topic by also being introduced to the JavaScript Object Notation (i.e. JSON), learn the advanced features of newer updates to the JavaScript programming language, and student how this programming language is used along with current web application frameworks that are used as the foundation to built modern dynamic web applications.

  1. Develop code that leverages Objects in a web page.
  2. Develop code that leverages Arrays in a web page.
  3. Develop code that handles Events in a web page..
  4. Explain what JSON is and how this data format is used to build web applications.
  5. Investigate Advanced Features found in recent updates to the JavaScript programming language
  6. Investigate recent JavaScript Web Application Frameworks that are available to build dynamic web applications

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Forms, JS Objects, JS Functions, and JS HTML DOM Tutorial sections.
  • W3Schools: Go to https://www.w3schools.com. Select the Learn JavaScript menu under the JavaScript section. Complete all reading and tutorials under the JS Browser BOM, JS AJAX, and JS JSON Tutorial sections.

Weeks 11-13

Topic 6 Web Development Libraries

In this topic students learn to use web development libraries to add dynamic, rich, and compelling content and user interfaces to their web pages. In this topic students are also introduced to the concept of responsive web pages, what this means, and how they can support this concept when building their web pages.

  1. Develop web pages that supports a Responsive Design.
  2. Develop web pages that leverages the use of jQuery that includes but is not limited to the use of Selectors, Events, DOM Manipulation, AJAX, and Data Table features
  3. Develop web pages that leverages the use of Bootstrap that includes but is not limited to the use of the Grid System and various UI Components (Buttons, Text, Alerts, Tables, Forms, and NavBar) features.

Readings

  • W3Schools: Go to https://www.w3schools.com. Select the Learn CSS menu under the HTML and CSS section. Complete all reading and tutorials under the CSS Responsive and CSS Grid Tutorial sections.
  • W3Schools: Go to https://www.w3schools.com. Select the Learn jQuery menu under the JavaScript section. Complete all reading and tutorials under the jQuery Tutorial, jQuery Effects, jQuery HTML, and jQuery AJAX Tutorial sections.
  • W3Schools: Go to https://www.w3schools.com. Select the Learn Bootstrap menu under the HTML and CSS section. Complete all reading and tutorials for the latest version of Bootstrap under the Bootstrap Tutorial and Bootstrap Grid Tutorial sections

Weeks 14-15

Topic 7 Advanced Web Development

In this topic students learn that building static web pages have their limits and that they often need to be deployed on web servers sometimes backed with a server-side programming language or database. This topic at a minimum will introduce the students to the HTTP protocol, client server model, HTML forms, and how to implement forms processing to future expand on the capabilities of the web pages they have built so far in previous topics. This topic introduces the students to the basic concepts required for future web application development courses where formal dynamic web application frameworks are used.

  1. Investigate the HTTP protocol and client server model.
  2. Develop web pages with HTML Forms.
  3. Develop web pages that integrate Forms Processing on the server.

Readings
